Párrafo uno. Una Página Web es un documento HTML. Puede ser estática (sólo código) o dinámica (con, por ejemplo, contenido extraído de una base de datos).

Párrafo dos. Un Sitio Web es un conjunto de páginas Web estructuradas en un dominio. Se puede crear escribiendo código o mediante CMS o builders., Pertenece a empresas, organizaciones o personas.
